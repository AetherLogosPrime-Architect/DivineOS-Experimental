## Claim

Cognitive-named CLI commands (ask, recall, decide, learn, feel, claim, opinion, compass-ops observe, mansion council) point at cognitive work but are not the work itself. Running the tool does not perform the cognition.

## Success criterion

Sessions that invoke a cognitive-named tool produce evidence of the cognition the tool's name claims (real consultation, real recall, real decision-making, real learning) rather than substituting tool invocation for the underlying mental act.

## Falsifier

If audit reviews surface sessions where the tool was invoked but the corresponding cognitive work was absent (e.g. 'ran ask' with no integration of returned content into next action; 'ran decide' with no actual evaluation of alternatives), the substitution pattern is alive and the bracketed [tool] labels alone are not sufficient defense.
